# LOCKDOWN — Submission Prompts (3-Image System)

Each submission uses three images combined in-game:
- **Attack Face** — character portrait, intense/finishing expression (generated ONCE per character, reused for all submissions)
- **Defense Face** — character portrait, pain/strain expression (generated ONCE per character, reused for all submissions)
- **Technique Shot** — close-up of the submission mechanic, grips and body parts only (one per submission, reusable across all pairings)

---

## TOTAL IMAGES NEEDED

| Type | Count |
|------|-------|
| Character face shots (2 per character) | 2 × number of characters |
| Technique shots (1 per submission) | 7 |

For a 2-character pair like Marcus + Adele: **4 face shots + 7 technique shots = 11 images total**, reused across all 7 submissions.

---

## HOW TO USE

**Face shots:** Upload the relevant character's `.png` and `Face Sheet.png` only. No position reference photos needed. Generate once, reuse everywhere.

**Technique shots:** No character reference images needed — just 2–3 reference photos of the technique. Generate once per submission — can be reused across all character pairings.

**One new ChatGPT conversation per image. Never reuse a conversation between images.**

**Save naming convention:**
- `[CHARACTER] Attack Face.png`
- `[CHARACTER] Defense Face.png`
- `[Submission] Technique.png`
