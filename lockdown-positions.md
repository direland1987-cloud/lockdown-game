# LOCKDOWN — Position Prompts (Character-Agnostic)
## 9 Positions + 7 Submissions | 16 Images Per Character Pair

---

## HOW TO USE

**Step 1 — Choose your character pair** from `lockdown-characters.md`.

**Step 2 — Decide who is FIGHTER_A and FIGHTER_B** for this image (A is listed first / dominant/attacker in each prompt).

**Step 3 — Before pasting the prompt into ChatGPT, upload:**
1. `[FIGHTER_A].png`
2. `[FIGHTER_A] Face Sheet.png`
3. `[FIGHTER_B].png`
4. `[FIGHTER_B] Face Sheet.png`
5–7. **3 Google Images results** for the position reference search query listed above each prompt

**Step 4 — Replace all placeholders in the prompt:**
- `[FIGHTER_A]` → Fighter A's name (e.g. Marcus)
- `[FIGHTER_B]` → Fighter B's name (e.g. Adele)
- `[FIGHTER_A CHARACTER DESCRIPTION]` → paste Fighter A's description block from `lockdown-characters.md`
- `[FIGHTER_B CHARACTER DESCRIPTION]` → paste Fighter B's description block from `lockdown-characters.md`

**One new ChatGPT conversation per image. Never reuse a conversation between images.**
