## How to Generate

1. Open ChatGPT (GPT-4 with image generation)
2. Copy each prompt above one at a time
3. If the image doesn't match (wrong person on top, wrong colors), regenerate or tell ChatGPT: *"The blue fighter needs to be on top, the orange fighter on bottom"*
4. Download each image and save with the filename listed below
5. **Place all 8 images into `raw_sprites/`** (same folder as the A1-A8 files)

## After You Have the Images

Run the processing pipeline:
```bash
node scripts/process_sprites.js --mode ground
node scripts/inject_scenes.js
```

This will:
1. Process them (remove backgrounds, trim, resize to game dimensions, convert to WebP)
2. Generate `processed/grapple_scenes.js` with base64 data
3. Inject the scene data into `lockdown_game.html`
4. The game will automatically use the correct artwork based on who's on top
